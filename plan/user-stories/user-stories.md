# User Stories

## Maya — Casual Player

### Story 1
As Maya, I want to start playing instantly without creating an account so that I can enjoy the game during short breaks.

### Story 2
As Maya, I want a simple and easy-to-understand interface so that I can play without needing instructions.


## Darren — Themed Player

### Story 3
As Darren, I want a visually cohesive and themed slot game so that I feel immersed while playing.

### Story 4
As Darren, I want to experience bonus rounds or special features so that I have something exciting to work toward.


## Priya — Social Player

### Story 5
As Priya, I want to easily share interesting or funny game results so that I can show them to my friends.
